Embeddable Activity Collector

Embedded collection, management and delivery of activity information from within the service execution environment.

Event Processor Network

Define networks of event processors which can be used to (a) analyse activity events, (b) apply business policies, (c) identify situations of interest and (d) derive new information which can be further analysed by other network nodes.

Active Collections

Information derived from the activity events can be made accessible to external consumers through an active collection mechanism - similar to standard collection but enable notifications to be emitted when the contents of the collection changes.

Analytic support using Kibana/Elasticsearch

"Elasticsearch is a flexible and powerful open source, distributed, real-time search and analytics engine." Starting with RTGov 2.0, activity information will be stored in Elasticsearch and analysed using the Kibana dashboard integrated with the RTGov UI.
